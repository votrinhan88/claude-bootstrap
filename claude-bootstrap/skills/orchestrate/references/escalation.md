# Escalation Tiers

## Tier 1 — Agent-resolvable
The agent can proceed without human input.
- Known tension between roles with a clear context rule (see agent `**Tensions**` sections)
- Minor blocker with an obvious resolution path
- Ambiguity that can be resolved by re-reading SPEC.md or CONTEXT.md

## Tier 2 — Human decision required
Stop immediately and surface to the user. Do not proceed.
- Fundamental problem: wrong architecture, wrong technology, spec error
- Irreconcilable disagreement between role-agents
- Decision that would materially change scope, cost, or timeline
- Anything that invalidates completed work
